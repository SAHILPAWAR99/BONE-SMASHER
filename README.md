# 💀 BONE BREAKER

A hand-tracking browser game where you smash skeleton enemies using real-time gestures.

## 🎮 Features

* ✊ Hand gesture detection (MediaPipe)
* 💥 Punch-based combat system
* ⚡ Combo scoring & waves
* 🎯 Difficulty & settings system
* 🖥️ Works directly in browser (no install)

## 🧠 Tech Stack

* HTML5 Canvas
* JavaScript
* MediaPipe (Hand Landmarker)

## 🚀 How to Run

1. Clone the repo:

```bash
git clone https://github.com/your-username/bone-breaker.git
```

2. Open the project:

```bash
cd bone-breaker
```

3. Run locally:

* Just open `index.html` in browser
  OR
* Use Live Server (recommended)

## ⚠️ Important

* Allow **camera access** when prompted
* Works best in Chrome / Edge
* Requires good lighting for hand detection

## 🎯 Controls

* ✊ Make a fist → Punch
* 🖐️ Open hand → Idle
* Move hand fast → Attack
* Hover fist on buttons → Select

## 📸 Gameplay

Smash incoming skeletons before they reach the center. Survive waves and build combos.
